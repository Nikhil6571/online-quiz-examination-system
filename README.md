# Online Quiz & Examination System

An interactive web-based Online Quiz & Examination System developed using **HTML, CSS, and JavaScript**.  
The application allows users to attempt MCQ-based quizzes with a fixed timer, automatic evaluation, randomized questions, and score history tracking.
